# Content type support

This reference compares user-facing capabilities that can meaningfully apply across content types. Features inherent to one kind of media, such as video stream or subtitle selection, are documented with that media's experience instead of being shown as unavailable for other types.

**✓** Supported &nbsp; **—** Not available

## Entry capabilities

| Capability | Manga | Anime |
| --- | :---: | :---: |
| Merge versions from different sources | ✓ | ✓ |
| Bulk-download child items | ✓ | ✓ |
| Bookmark individual child items | ✓ | — |
| Filter child items by release group | ✓ | — |
| Migrate an entry to another source | ✓ | — |
| Delete downloads after marking an item consumed | ✓ | — |

!!! note

    Bookmark-based bulk downloads are available when the content type supports individual bookmarks.
